## 简介

AOP 全称为 Aspect-oriented programming，很明显这是相对于 Object-oriented programming 而言。Aspect 可以翻译为“切面”或者“侧面”，所以 AOP 也就是面向切面编程。

AOP 做一个浅显的总结就是在不修改原代码的情况下,动态添加功能的一种技术.
